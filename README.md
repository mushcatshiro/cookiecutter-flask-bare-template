# flask-toolbox (bare template)

to provide a single file template for web application (static content, changing it into js front end framework shouldnt be too difficult).

## what's available?

- celery
  - may change to rq instead
- task status and result dashboard
- base html template with alert message
- d3.js for data visualization
  - replaced by echarts for simplicity
- database (sqlalchemy)
